Student Performance & Attendance Dashboard
Overview

This dashboard provides a visual analysis of student academic performance and attendance across departments, subjects, and semesters. It helps identify performance trends, subject difficulty, and departmental attendance patterns for data-driven academic decision-making.

Key Metrics Displayed

Average Attendance: 86.22%

Average Internal Test 1 Marks: 66.36

Average Internal Test 2 Marks: 64.48

Average Assignment Marks: 14.06

These KPIs give a quick snapshot of overall student performance.

Visualizations Included
1. Average Internal Test Marks by Department and Subject

Compares Internal Test 1 and Internal Test 2 scores

Grouped by Department (AI&DS, CSE, IT) and Subject

Helps identify subjects with lower or higher performance

2. Average Attendance by Semester and Department

Displays attendance trends across Semesters 5, 6, and 7

Department-wise comparison

Useful for monitoring student engagement

3. Sum of Internal Test Marks by Subject

Horizontal stacked bar chart

Shows combined contribution of Internal Test 1 and Internal Test 2

Enables subject-level performance comparison

Filters / Slicers

Department: AI&DS, CSE, IT

Semester: 5, 6, 7

These filters allow interactive analysis and focused insights.

Purpose of the Dashboard

Identify weak subjects and departments

Analyze test performance trends

Monitor attendance patterns

Support academic planning and teaching strategy improvements

Tools Used

Data visualization dashboard tool (e.g., Power BI / similar BI tool)

Academic dataset containing attendance, internal marks, and assignments
